入门

1. 安装

1.1. 服务器要求

  • 一个Web服务器，如Apache或nginx

  • 数据库

    • MySQL (版本>5.7.8) / MariaDB(版本>10.2)
    • PostgreSQL (版本>9.2)
    • Lyche的内置SQLite3支持

  • PHP>=7.4带有这此扩展：

    • BCMath
    • Ctype
    • Exif
    • Ffmpeg（可选 - 可以生成视频缩略图）
    • Fileinfo
    • GD
    • Imagick (可选 - 可以生成更好的缩略图)
    • JSON
    • Mbstring
    • OpenSSL
    • PDO
    • Tokenizer
    • XML
    • ZIP

Lychee强烈建议使用64位操作系统，虽然它可以在32位系统上工作
