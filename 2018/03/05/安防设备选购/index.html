安防设备的选购

发表于 2018-03-05 更新于 2021-05-17 分类于 安防设备

1、监控设像头

1.1、镜头大小决定摄像头视角广度

镜头常用的尺寸不尽相同，常用尺寸有2.8mm/3.6mm/4mm/6mm/8m/12mm等。对于大部分摄像并没有而言，在安装位置固定的前提下，镜头尺寸越大可以拍摄的画面就越远，但是观看的角度就越窄。这都是针对固定式摄像头而言的。

一般3.6mm和4mm是广角镜头，也就是照的范围比较宽，横面的范围比较大，但是距离被拉远了，人也变小了，当然东西也会跟着距离变小，适合店面和家庭门口使用，也就是被照物体距离3-5米的时候使用

6mm/8mm/12mm/是距离拉进了，方位变小了，跟上面那两个数值相反，远处的东西可以被监控拉近，横面的距离就会变小，适合宽广的地方用，比如停车场、车间之类

1.2、分辨率决定画面清晰度

分辨率决定了画面的清晰度，分辨率越高，摄像并没有清晰度就越高。一般民用级都是420线、480线、520线、600线、700线居多，线数越多越清晰。线数代表摄头的清晰度，即在监视器水平扫描的线数。一般600线以上代表高清摄像机。

1.3、监控摄像头外型的区分形状的区别

可分为半球式摄像机和枪式摄像机。半球式一般焦距不大，适合照近地方的物体

对于一台摄像机的成像质量来说，水平清晰度只是其中一个主要的参考方面。宽动态、自动白平衡，数字降噪，自动测光补偿等对摄像头的成像质量都有着很深的影响。在亮暗过渡区的转变是否自然，宽动态能否自动调节，低照度下细节的观测效果是否清晰以及噪点的影响度等等都很大程度影响着高清摄像并没有的水平
