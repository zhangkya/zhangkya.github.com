Clover中隐藏“Boot Windows From EFI”启动项

用Clover时经常能碰到Boot Windows From EFI或者Boot Grub From EFI等启动项，又启动不起来Windows或者Ubuntu。
经过小小的研究，修改Clover的config.plist文件，将GUI->Scan下的Legacy选项设为false即可隐藏。
